## 定位和选择
### 直达
- Ctrl+d x
- Ctrl+d a(增加) c(配置) d(去除)

- alt+f1 x 切换磁盘
- alt+f2 x 切换右窗口

- ctrl+左/右 窗口目录复制到一边
- alt+s 快速命令(可以配置任意命令)
- alt+左/右 目录历史记录
- 字母直接搜索(需要配置选项)

-alt+f10 文件夹树

### 标签
- ctrl + t 新建
- ctrl + w 关闭
- ctrl + shift + w 关闭所有非活动的
- ctrl + tab 切换
- ctrl + shift + tab 切换
- ctrl + 上 把光标上的目录在新tab 打开

### 压缩以及插件
- alt + f5 解压
